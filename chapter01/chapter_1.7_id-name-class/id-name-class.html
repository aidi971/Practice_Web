class
用于定义类名，一般该类名就是指向CSS样式表中定义的类
那么，既然css中的类可以被定义于标签，
我们就可以通过javascript脚本语言对其进行增加、修改、删除
从而实现动态改变样式类的功能。
但是，不是全部标签都可以使用class属性，如：
base、html、head、script、style、title等标签不支持该属性
